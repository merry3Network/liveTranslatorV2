# セットアップガイド (まずはここから)

`command not found: npm` というエラーが出る場合、お使いのパソコンに **Node.js** がインストールされていません。
以下の手順に従って環境を整えてください。

## 1. Node.js のインストール
以下のいずれかの方法でインストールしてください。

### 方法 A: 公式インストーラーを使う (一番簡単)
1. [Node.js 公式サイト](https://nodejs.org/ja/) にアクセスします。
2. **「推奨版 (LTS)」** と書かれたボタンをクリックしてインストーラーをダウンロードします。
3. ダウンロードしたファイルを開き、画面の指示に従ってインストールを完了させます。

### 方法 B: Homebrew を使う (エンジニア向け)
ターミナルで以下のコマンドを実行します。
```bash
brew install node
```

---

## 2. 準備と設定
ターミナルを開き、プロジェクトフォルダに移動して設定を行います。

```bash
# 1. フォルダへ移動
cd /Users/rsk/Dev/Projects/Apps/LiveTranslator

# 2. 依存ライブラリのインストール
npm install

# 3. 環境設定ファイル (.env) の作成
cp .env.example .env
```

その後、`.env` ファイルを開き、必要な API キーなどを入力してください。

---

## 3. ローカルAI (Ollama) のセットアップ (任意)
外部APIを使わず、自分のPCの力だけで翻訳したい場合は以下の手順を行います。

1. [Ollama 公式サイト](https://ollama.com/) からアプリをダウンロード。
2. ターミナルで推論モデルをダウンロード（Intel Macなら `gemma2:2b` がおすすめ)
   ```bash
   ollama pull gemma2:2b
   ```
3. `.env` の以下の項目を書き換えます：
   ```env
   ENABLE_LOCAL_AI=true
   OLLAMA_MODEL=gemma2:2b
   ```

---

## 4. アプリの起動
アプリを動かすには **「サーバー (バックエンド)」** と **「画面 (フロントエンド)」** の両方を立ち上げる必要があります。

**手順 A: 一括起動 (推奨)**
1つのターミナルで以下を実行します。
```bash
npm run dev
```

**手順 B: 個別に起動する場合**
修正作業などを行う際はこちらが便利です。

- **ターミナル 1 (サーバー用)**
  ```bash
  npm start
  ```
- **ターミナル 2 (画面用)**
  ```bash
  npm run dev:frontend
  ```

起動後、ブラウザで `http://localhost:5173` を開いてください。
接続が成功し、マイクの使用を許可すれば翻訳が開始されます！
